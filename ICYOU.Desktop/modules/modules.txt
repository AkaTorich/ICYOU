Папка для модулей.

Поместите скомпилированные DLL модулей в эту папку.
Модули будут автоматически загружены при старте сервера/клиента.

Для создания модуля:
1. Создайте проект библиотеки классов .NET 8.0
2. Добавьте ссылку на ICYOU.SDK
3. Создайте класс, наследующий ModuleBase или реализующий IModule
4. Скомпилируйте и поместите DLL сюда
