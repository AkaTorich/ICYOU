Этот файл - placeholder для папки смайлов.

Чтобы добавить смайлы:
1. Поместите файлы картинок (.gif, .png, .jpg) в эту папку
2. Имя файла без расширения станет кодом смайла

Примеры:
- :).gif -> код :)
- =).gif -> код =)
- smile.png -> код smile
- heart.gif -> код heart

Поддерживаемые форматы: gif, png, jpg, jpeg, webp
